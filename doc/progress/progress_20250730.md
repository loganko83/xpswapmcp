### 1. Math.random() 취약점 수정 (치명적)

- **수정된 파일들**:

  - `tests/setup.ts`

**수정 내용**:
```javascript
// 이전 (취약함)
Math.random()

// 이후 (안전함)
(crypto.randomBytes(4).readUInt32BE(0) / 0xFFFFFFFF)
```
